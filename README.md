# Click2Buy GTM Server-Side Tag Template

This repository contains the official Google Tag Manager Server-Side Tag Template for Click2Buy conversion tracking.

This template allows retailers to track conversions referred by the Click2Buy platform using a robust, first-party cookie mechanism.
